# Getting Started

This is a demo of remix + react + shadcn + storybook.

## Local Development

Run the dev server:

```sh
npm run dev
```

## Local Build

First, build your app for production:

```sh
npm run build
```

Then run the app in production mode:

```sh
npm start
```

## Storybook

```sh
npm run storybook
```

## Presenting Storybook

```sh
npm run build-storybook
cd storybook-static
python3 -m http.server
```
